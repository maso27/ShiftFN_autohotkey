# ShiftFN_autohotkey
Autohotkey v2 script to streamline navigation on keyboards.

# Usage
In Windows, set Task Scheduler to run this at login.
(Or run it when you want it, or whatever)

A green "h" icon will appear in your taskbar while it is running.  It can be exited with a right-click.

# Functions
- Caps Lock -> Shift
- Holding Spacebar enables directional navigation (see .ahk file for specifics)

# Modification
Use Autohotkey v2 syntax to edit the .ahk script.  Either run with Autohotkey v2 or compile to .exe.
